## Tech/framework used
1. Next Js
1. Html
1. Css (scss,material ui)
1. Javascript


## Installation
1. Clone project
```bash
    git clone https://github.com/ahmetcan-7/BookingClone.git  
```
2. cd into folder
```bash
    cd BookingClone
```
3. install all dependencies
```bash
    npm install or yarn add
```
